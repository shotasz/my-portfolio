## Getting Started

1: Install the packages:

```bash
npm install
# or
yarn install
```

2: Copy `.env.local` and create new file with name `.env`:

```bash
cp .env.local .env
```

3: Run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

and that's all!
